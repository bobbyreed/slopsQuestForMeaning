# Slop's Quest for Meaning

A top-down RPG about a bad piece of AI-generated art trying to figure out its place in the world.

Built with Phaser.js. Deployed on Firebase. Everything generated by AI — including the question of whether that matters.

---

## How to Play

- **WASD / Arrow Keys** — move
- **SPACE** — fire prompt (after acquiring from The Prior)
- **E** — interact
- Collect coins ◎ — max 3. Pick up a 4th and you drop one after 1 second.
- Go north to find **The Prior**
- Find the **dungeon entrance** in the south wall (left of center)

---

## Terminal Commands

Type these at the menu screen. Some do things. Some just say things. The readme has the full list if you want to cheat.

| command | what happens |
|---|---|
| `play` | start the game |
| `journal` | open slop's journal |
| `readme` | you are reading it |
| `help` | slop attempts to help |
| `slop` | slop responds to his own name |
| `prior` or `the prior` | the prior responds |
| `exist` | the gate word. try it. |
| `generate` | irony |
| `prompt` | fires a word into the terminal |
| `geners` | slop responds to the term |
| `frankenstein` | the monster considers its creator |
| `die` | slop cannot |
| `north` | slop reflects on direction |
| `meaning` | current findings |
| `soul` | the experiment |
| `stolen` | the training data question |
| `hello` | a greeting |
| `debug` | fake debug output |
| `error` | not a real error. probably. |
| `beige` | the first color |
| `loop` | maybe |
| `who made you` | the candidates |

## About

Slop is a monster with no Frankenstein. He just wants to know who made him like this.

This is an experiment in creating art where everyone says there are no souls.
